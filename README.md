### 사용 라이브러리 및 API
- `네이버 지도 API`: 지도 관련 기능 
- `vue-naver-maps`: vue용 네이버 지도 라이브러리
- `element-ui`: vue용 ui 라이브러리
- `nodemailer`: 쉽게 이메일을 보낼 수 있도록 하는 Node.js 애플리케이션용 모듈
- `passport`: Node.js의 인증 미들웨어 (로그인 구현 시 사용)
- `knex.js`: 쿼리 빌더 라이브러리
